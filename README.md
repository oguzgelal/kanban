# Kanban Board

Built with Lagom Framework.

### Build & Run

```
sbt build:runAll
```
Should start running on http://localhost:9000

### Run

```
sbt runAll
```
Should start running on http://localhost:9000

### Running Unit tests

Unit tests for **task service**

```
sbt task-impl/test
```
Unit tests for **board service**

```
sbt board-impl/test
```
